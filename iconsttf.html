Cittadino:62 TTF Icons

BY Dmitry Goloub License
All of the icons are available in TTF and SVG format. We are using our own selector dcon instead of icon to avoid conflict with other icons.
You can get access to all of the icons by placing icons-ttf.css file under main decorator.css file.
If you want to customize icons for example (change color or size), you can do so in your theme.css file as shown in example below:
.dcon {
     font-size: 21px;
     color: #555;
}
